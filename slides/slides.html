How to check if a number is prime using regex?

Like this?

def prime?(n)
  n.to_s.match? /^(2|3|5|7)$/
end

What about 11, 13 and others?

(0..100).select { prime?(_1) }
#=> [2, 3, 5, 7]

def prime?(n)
  '1'*n !~ /^1?$|^(11+?)\1+$/
end

(0..100).select { prime?(_1) }
#=> [2, 3, 5, 7, 11, (...), 89, 97]
